# 👟 StepX — Premium Sneaker E-Commerce UI

StepX is a modern, conversion-focused e-commerce frontend built to simulate a real-world sneaker store experience.

This project focuses on **UI/UX, performance, and user flow**, making it a strong foundation for a production-ready e-commerce platform.

---

## Deployment link:
https://step-xsneak.vercel.app/

## 🚀 Project Overview

StepX is a fully connected frontend application that demonstrates how a user moves from browsing products to completing a purchase.

It is designed to showcase:

* Real-world e-commerce structure
* Conversion-focused design
* Clean and scalable frontend architecture

---

## 🧠 Key Features

### 🛍️ Store UI

* Product listing (shop page)
* Product detail pages
* Add to cart functionality (frontend state)
* Cart management (update/remove items)
* Checkout UI (mock flow)

---

### 🔗 Connected User Flow

* Homepage → Shop → Product → Cart → Checkout
* All pages are properly linked
* Smooth navigation experience

---

### 📱 Responsive Design

* Mobile-first approach
* Fully responsive (mobile, tablet, desktop)
* Sticky CTA on mobile for better UX

---

### 🎨 UI/UX Design

* Premium dark theme (Nike-inspired)
* Clean, minimal layout
* Conversion-focused sections
* Trust elements and urgency indicators

---

### ⚡ Performance

* Optimized layout structure
* Lightweight components
* Fast loading UI

---

## 🛠️ Tech Stack

* Next.js (App Router)
* Tailwind CSS
* React (State Management for Cart)

## 🔄 User Flow

1. User lands on homepage
2. Browses products in shop
3. Views product details
4. Adds item to cart
5. Proceeds to checkout
6. Reviews order (mock checkout)

---

## ⚠️ Important Note

This project currently includes **frontend functionality only**.

* No backend integration yet
* No real payment processing
* Checkout is a simulated flow
